# Agent Skills

A collection of agent skills for GitHub Copilot, Claude, and other AI coding assistants.

## Skills

| Skill | Description |
|-------|-------------|
| [oidc-integration](./oidc-integration/) | Guidelines for integrating OIDC providers (e.g., Authing, IdentityServer, Keycloak) into frontend and backend applications. |

## Installation

```bash
npx skills add Jeff-Tian/agent-skills
```

Install a specific skill:

```bash
npx skills add Jeff-Tian/agent-skills --skill oidc-integration
```
